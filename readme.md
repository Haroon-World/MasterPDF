# MasterPDF

A Python desktop application for performing PDF operations.

## Features
- Merge PDF
- Split PDF
- Protect PDF
- Unlock PDF
- Compress PDF
- Extract Pages

## Technologies
- Python
- CustomTkinter
- pypdf

## Installation

pip install -r requirements.txt

python main.py

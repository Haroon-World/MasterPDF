# MasterPDF

A Python desktop application for performing PDF operations.

## Features
- Merge PDF
- Split PDF
- Protect PDF
- Extract Pages

## Technologies
- Python
- CustomTkinter
- pypdf
- PyMuPDF

## Installation

pip install -r requirements.txt

python main.py

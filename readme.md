# PDFMaster (v1.5)

Welcome to **PDFMaster** (Version 1.5) — a lightweight, clean, and efficient desktop utility built with Python and Qt for managing, converting, and editing PDF and image documents locally. 

This tool provides a straightforward, user-friendly interface for everyday document manipulation tasks without the need for cloud services or complex software.

## ✨ Features

* **Merge PDFs:** Combine multiple PDF documents into a single, organized file.
* **Split PDFs:** Extract specific pages or split multi-page documents into individual files.
* **Watermark PDFs:** Apply custom text watermarks across document pages to secure your work.
* **Compress PDF:** Reduce document file sizes efficiently for easy sharing and storage.
* **PDF to Image:** Convert individual pages or entire PDF files into high-quality image formats (PNG/JPG).
* **Image to PDF:** Combine multiple images seamlessly into a single compiled PDF document.
* **Rotate Pages:** Quickly adjust page orientations with intuitive rotation tools.
* **Live Visual Preview:** Built-in PDF and image viewer to inspect pages and layouts seamlessly before executing any operations.

## 🛠️ Technologies Built With

* **Language:** Python 3.x
* **GUI Framework:** PyQt6 (Qt)
* **PDF Engine:** PyMuPDF (`fitz`)

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

You will need **Python 3.8 or newer** installed on your machine. You can download it directly from [python.org](https://www.python.org/).

### Installation

1. **Clone the repository:**
   Open your terminal or command prompt and run:
   ```bash
   git clone [https://github.com/Haroon-World/MasterPDF.git](https://github.com/Haroon-World/MasterPDF.git)
   cd MasterPDF
2. Install The Required Dependencies
It is highly recommended to use a virtual environment. The project requires PyMuPDF and a Qt framework (like PyQt6). Install them using pip:

pip install -r requirements.txt
pip install PyMuPDF==1.24.1 PyQt6==6.7.0

3. Launch the application:
Start the application by running the main Python script:

python main.py
